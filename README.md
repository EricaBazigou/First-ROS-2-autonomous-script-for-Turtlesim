# 🏎️ ROS 2 F1 Autonomous Circle Driver

Αυτό είναι το πρώτο μου project στον προγραμματισμό αυτόνομων οχημάτων χρησιμοποιώντας το *ROS 2 Jazzy*.

## 📝 Περιγραφή
Ένας controller σε Python που καθοδηγεί ένα ρομποτικό όχημα (Turtlesim) να εκτελεί κυκλική τροχιά. Αυτό αποτελεί τη βάση για την κατανόηση της κίνησης και του ελέγχου σε περιβάλλοντα προσομοίωσης F1.

## 🛠️ Τεχνολογίες
- *OS:* Ubuntu 24.04 (WSL2)
- *Framework:* ROS 2 Jazzy Jalisco
- *Language:* Python 3

## 🚀 Πώς να το τρέξετε
1. Ξεκινήστε τον προσομοιωτή: ros2 run turtlesim turtlesim_node
2. Τρέξτε τον κώδικα: python3 circle_drive.py
